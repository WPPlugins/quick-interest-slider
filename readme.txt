A simple Interest Calculator. Uses slider to set the amount and the term and display a range of outputs

== Description ==

Ideal for payday loans, fixed fee payments, regular payments, savings, comparisons and pretty much anything else that gets calculated from a variable amount and term.

= Features =

*   Ultra lightweight - under 5kB
*   Any currency
*   Any interest Rate
*   Set the max, min, initial and step values on the sliders
*   Set the term to days, weeks or months
*   Select from a range of different outputs
*   Change the labels on all outputs
*   Style borders, colors and backgrounds
*   Pro version with application form and autoresponders

== Installation ==

1.	Download the plugin
2.  Login to your wordpress dashboard.
3.	Go to 'Plugins', 'Add New' then upload.
4.	Follow the on screen instructions.
5.	Activate the plugin.
6.	To use the plugin in your posts and pages add the shortcode `[qis]`.
7.	Change the options in the plugin settings page.
